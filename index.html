Game Instructions

Connect 4 is a two-player game where each player takes turns dropping colored discs into a vertical grid. Here are the instructions:

  • The goal is to be the first to form a horizontal, vertical, or diagonal line of four of one's own discs.
  • The game ends when one player achieves this or the grid is completely filled, resulting in a draw if no line is formed.
  • You can adjust the bot's parameter *using the sliders*
